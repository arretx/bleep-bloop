The basic process is this:

1. Create a list of messages.
1. Store those message in an array.
1. Count those messages.
1. Generate a random number based on that count.
1. Get the message at the random number's index position in the array.
1. Make that message be the payload.
1. Use that payload in a notification as the message data.
